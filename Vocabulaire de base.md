# Vocabulaire de base

Condamnation
Je = I
Le = the
Veux = want
Avoir = get
Ça = that

Je le veux. Je veux ça. Je veux l’avoir. Je veux avoir ça.

Voudrais = would like
Voir = see

Je voudrais la voir.

Dois = must
Parler = speak
Avec = with
Elle = she
Lui = him

Je dois parler avec elle-lui.

Aime = to love
Faire = make

J’aime le faire.

Y = to it; there
Aller = go

Je voudrais y aller.

Vais = will

Je vais y aller.

Mais = but
Trop, très = very
C’est = it’s
Loin = long route
Ne … pas = not
Ce = it
N’est = isn’t

C’est très loin d’ici. C’est loin d’ici? Ce n’est pas loin.

Près = near

C’est près d’ici.

À = on, in
Pouvez = can
Vous = you

Vous pouvez aller à pied.

Prendre = to take

Vous pouvez prendre un taxi.

Ça va = it goes
Bien = good

C’est bien. Ça va bien.
